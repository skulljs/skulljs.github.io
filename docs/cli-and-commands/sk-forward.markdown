# sk forward

## Feature

Forward command to the projects CLI. Ex: nest for nestjs / ng for angular

## Usage

{% highlight bash %}
$ sk ng serve
$ sk nest start
{% endhighlight %}

## Example

{% highlight bash %}
$ sk nest start
{% endhighlight %}
